 Python Inventory Management System
This project is a simple Inventory Management System written in Python.
It helps manage products using lists, loops, conditionals, and user input handling.
:white_check_mark: Features
Add Product (name, price, stock)
Remove Product (with error message if not found)
Edit Product
Update price
Update stock quantity
Show All Products
Search Product by name
Calculate Total Inventory Value
Buy Product (updates stock automatically)
Menu-driven system with continuous interaction
:pushpin: What I Practiced in This Project
Working with lists of lists
Using loops and conditions
Handling user input
Updating values inside lists
Implementing menu-driven programs
Simple algorithmic thinking
:open_file_folder: Project Purpose
This project was created to strengthen my understanding of Python fundamentals through real-world mini projects.
